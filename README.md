Upward Tests
============

This repository contains tests for [Upward](https://www.github.com/rtm/upward), the declarative Javascript library.


### Building and running

    git clone git@github.com:rtm/upward-test.git
    cd upward-test
    make build
    make serve

Then visit `localhost:8081`.
